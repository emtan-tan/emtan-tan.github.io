Let's Get Started!

The first example we should look at is a core menu item and we will use it as a reference point for simple beverages: Venti Iced Blonde Vanilla Latte. Let's break it down!

  • Venti: This is the size and the best piece to start with in order for your barista to select the correct cup
  • Iced: This is an equally important piece for the same reason but also in case ingredient proportions are different.
  • Blonde: This is defining the type of espresso we will be using in your beverage, this step can be skipped but it is vital if you are interested in blonde or decaf espresso.
  • Vanilla: This is the flavor of the drink, a simple but important part.
  • Latte: This is the base drink type, it'll define how many shots we use, how many pumps we put in, milk to foam ratio, and many other things for other types of beverages!

This is exactly how the barista has learned to recite the order when calling it out for a customer! This can reduce confusion and wait times for everyone!

Let's Keep Going! You're Doing Great!

There are, of course, many more beverages as well as many more modifications! For the moment though, let's take a look at the different options available for these select modifiers.

Size

There are a few size options available at Starbucks:

  • Hot:
    • Kids (8oz)
    • Short (8oz)
    • Tall (12oz)
    • Grande (16oz)
    • Venti (20oz)
  • Iced
    • Tall (12oz)
    • Grande (16oz)
    • Venti (24oz)
    • Trenta (30oz)

Craft Type

There are not too many options here but the main ones are:

  • Hot
  • Iced
  • Blended

Espresso Type

This one also has few options but the primary three are:

  • Signature
  • Blonde
  • Decaf

Flavor

There are a vast and large amount of flavors and a rotating menu surrounding those, a basic list consists of:

  • Mocha
  • White Mocha
  • Peppermint
  • Vanilla
  • Sugar-Free Vanilla
  • Hazelnut
  • Cinnamon Dolce
  • Brown Sugar
  • Caramel
  • Sugar-Free Caramel

Build Type

The base type of the drink is going to dictate lots of things if they are not otherwise modified like a latte is espresso and steamed milk or an Americano is espresso and water. Primary basic examples for Espresso beverages are:

  • Latte
  • Flat White
  • Macchiato
  • Americano
  • Cappuccino
  • Cortado

Extra Modifiers

This is where it can get fun but also complicated! You get the choice to modify your beverage exactly how you want. For espresso beverages, common modifiers are changing the:

  • Amount of foam
  • Whipped cream
  • Temperature of the milk
  • Type of milk
  • Amount of room
and that's just the beginning!

For non-espresso drinks, this portion can include things like powders, inclusions, toppings, cold foams, or custom directions like stirred or in a larger cup.

Here's Some More Examples!

There are lots of more beverages available and let's look at examples of each type!

Venti Iced Brown Sugar Oatmilk Shaken Espresso

This is an example of a shaken espresso beverage, it has the following modifiers:

  • Size: Venti
  • Craft Type: Iced
  • Espresso Type: Blonde
  • Flavor: Brown Sugar
  • Milk Type: Oatmilk
  • Build Type: Shaken Espresso
If the drink default comes with a certain modifier, you can omit saying it. So for this drink, we could just say "Venti Iced Brown Sugar Oatmilk Shaken Espresso" and the barista would know to use blonde espresso since that is the default for this beverage, but if we wanted to change it to decaf, we would need to say "Venti Iced Decaf Brown Sugar Oatmilk Shaken Espresso."

Grande Iced Green Tea Lemonade with 2 Pumps of Classic

This is an example of an iced tea beverage, it has the following modifiers:

  • Size: Grande
  • Craft Type: Iced
  • Build Type: Green Tea Lemonade
  • Extra Modifier: 2 Pumps of Classic Syrup
As you can see, other drinks can be simpler and contain different modifiers than espresso beverages, but the same rules apply! If you wanted to change the flavor, you would just need to say the new flavor in place of the old one. So if we wanted to change it to raspberry, we would say "Grande Iced Raspberry Green Tea Lemonade with 2 Pumps of Classic."

Trenta Vanilla Sweet Cream Cold Brew with Light Ice and Caramel Drizzle

This is a cold brew beverage, it has the following modifiers:

  • Size: Trenta
  • Flavor: Vanilla
  • Milk Type: Sweet Cream
  • Build Type: Cold Brew
  • Extra Modifier: Light Ice
  • Extra Modifier: Caramel Drizzle
Again, the same rules apply! If we wanted to change the milk type to oatmilk, we would say "Trenta Vanilla Oatmilk Cold Brew with Light Ice and Caramel Drizzle." Since it is a cold brew, we would not need to say the craft type since it is implied by the build type being Cold Brew.

Tall Hot Blonde Cinnamon Dolce Flat White with 3 Splendas Stirred

Here is another espresso beverage example, it has the following modifiers:

  • Size: Tall
  • Craft Type: Hot
  • Espresso Type: Blonde
  • Flavor: Cinnamon Dolce
  • Build Type: Flat White
  • Extra Modifier: 3 Splendas
  • Extra Modifier: Stirred
If we wanted to change the flavor to hazelnut, we would say "Tall Hot Blonde Hazelnut Flat White with 3 Splendas Stirred."

Nonfat Pistachio Cortado

This is a more unique example, here are the following modifiers:

  • Milk Type: Nonfat
  • Flavor: Pistachio
  • Build Type: Cortado
The default size for a cortado is always a short (8oz) and the default craft type is hot, so we do not need to say those modifiers. It is made this way intentionally so the quantity of coffee is equal to the quantity of milk.
