Let's Get Started!

The first example we should look at is a core menu item and we will use it as a reference point for simple beverages: Venti Iced Blonde Vanilla Latte. Let's break it down!

  • Venti: This is the size and the best piece to start with in order for your barista to select the correct cup
  • Iced: This is an equally important piece for the same reason but also in case ingredient proportions are different.
  • Blonde: This is defining the type of espresso we will be using in your beverage, this step can be skipped but it is vital if you are interested in blonde or decaf espresso.
  • Vanilla: This is the flavor of the drink, a simple but important part.
  • Latte: This is the base drink type, it'll define how many shots we use, how many pumps we put in, milk to foam ratio, and many other things for other types of beverages!

This is exactly how the barista has learned to recite the order when calling it out for a customer! This can reduce confusion and wait times for everyone!

Let's Keep Going! You're Doing Great!

There are, of course, many more beverages as well as many more modifications! For the moment though, let's take a look at the different options available for these select modifiers.

Modifier Primary Options Other Notes
Size There are a few size options available at Starbucks:
Hot: Kids (8oz), Short (8oz), Tall (12oz), Grande (16oz), or Venti (20oz)
Iced: Tall (12oz), Grande (16oz), Venti (24oz), or Trenta (30oz) 		Kids is the same size as short but that size option is usually used for a kids hot chocolate and it is done at a lower temperature for the kid.
Craft Type There are not too many options here but the main ones are:
Hot, Iced, or Blended. 		Blended is not a common one to define since most of those drinks automatically have it in the recipe like the Frappuccino or drinks deemed "frozen." Additional modifiers that can be used here are extra hot or double blended.
Espresso Type This one also has few options but the primary three are:
Signature, Blonde, and Decaf 		Signature is the default option, so it is not necessary to state it, but it is greatly important for decaf or blonde! Other options are 1/2, 1/3, or 1/4 decaf depending on how many shots are in a drink.
Flavor There are a vast and large amount of flavors and a rotating menu surrounding those, a basic list consists of: mocha, white mocha, peppermint, vanilla, sugar-free vanilla, hazelnut, cinnamon dolce, brown sugar, caramel, sugar-free caramel, and many, many more. When reciting your flavor, you can also define the amount of pumps you want if you desire to change it from default. So you can order a "Venti hot 2 pump vanilla latte."
Build Type The base type of the drink is going to dictate lots of things if they are not otherwise modified like a latte is espresso and steamed milk or an Americano is espresso and water. Main examples are: Latte, Flat White, Macchiato, Americano, or Cappuccino for espresso beverages. There are also many other build types like hot chocolates, cold brews, iced teas, iced coffees, refreshers, shaken espressos, and many more!
Extra Modifiers This is where it can get fun but also complicated! You get the choice to modify your beverage exactly how you want. For espresso beverages, common modifiers are changing the: amount of foam, whipped cream, temperature of the milk, type of milk, amount of room, and that's just the beginning! For non-espresso drinks, this portion can include things like powders, inclusions, toppings, cold foams, or custom directions like stirred or in a larger cup.
